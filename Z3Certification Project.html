Introduction

HTML (HyperText Markup Language) is the standard markup language used to create web pages. HTML documents are made up of a series of tags, which tell the browser how to display the content. HTML tags are enclosed in angle brackets, like this: <tagname> . Most HTML tags come in pairs: an opening tag, like <html>, and a closing tag, like </html>. The text between the opening and closing tags is the content of the element.

HTML5 is the latest version of HTML and is widely used for web development. In this document, we'll cover the basics of HTML5, including the most commonly used tags and attributes.

  • Creating web pages: HTML is the backbone of web pages. It defines the structure and content of a webpage, including headings, paragraphs, images, links, and more.
  • Creating forms: HTML is used to create forms that allow users to submit information, such as login credentials or contact information.
  • Displaying multimedia: HTML allows developers to embed multimedia, such as images, videos, and audio files, on a web page.
  • Creating tables: HTML can be used to create tables to organize data in a structured format.
  • Enhancing accessibility: HTML can be used to improve accessibility by including semantic markup that helps screen readers and other assistive technologies understand the content of a web page.

Basic Structure

The basic structure of an HTML document includes the doctype, head, and body elements. The doctype is used to tell the browser which version of HTML is being used, the head element contains metadata about the document, and the body element contains the content of the document.

Example

<!DOCTYPE html>
<html>
<head>
<title>My Web Page</title>
</head>
<body>
<h1>Welcome to my web page</h1>
<p>This is the content of my web page</p>
</body>
</html>

Headings

Headings are used to organize content into sections, and they range in size from H1 to H6. The H1 element is used for the main heading of the page, while the H2 to H6 elements are used for subheadings.

Example:

<h1>Main Heading</h1>
<h2>Subheading 1</h2>
<h3>Subheading 2</h3>

Paragraphs

The P tag is used to create paragraphs of text. Paragraphs are the most basic way to display blocks of text in HTML.

Example:

<p>This is a paragraph of text.</p>

Links

Links are used to connect web pages together or to other types of content. The A tag is used to create links, and the HREF attribute is used to specify the URL of the link.

Example:

<a href="http://www.example.com">Link Text</a>

Images

The IMG tag is used to insert images into web pages. The SRC attribute is used to specify the URL of the image, and the ALT attribute is used to provide a text description of the image for users who cannot see it.

Example:

<img src="example.jpg" alt="Example Image">

Lists

There are two types of lists in HTML: ordered and unordered. The OL tag is used to create an ordered list, while the UL tag is used to create an unordered list. List items are created using the LI tag.

Example:

<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>

Forms

Forms are used to collect information from users. The FORM tag is used to create a form, and various input elements (such as text boxes, checkboxes, and radio buttons) are used to collect information from users.

Example:

<form action="submit.php" method="post">
<label for="name">Name:</label>
<input type="text" id="name" name="name"><br>
<label for="email">Email:</label>
<input type="email" id="email" name="email"><br>
<input type="submit" value="Submit">
</form>

Tables

The TABLE tag is used to create a table, and the TR tag is used to create table rows. TD tags are used to create table data cells, and TH tags are used to create table header cells.

Example:

<table>
<tr> <th>Header 1</th>
<th>Header 2</th>
<th>Header 3</th>
</tr>
<tr>
<td>Row 1, Column 1</td>
<td>Row 1, Column 2</td>
<td>Row 1, Column 3</td>
</tr>
<tr>
<td>Row 2, Column 1</td>
<td>Row 2, Column 2</td>
<td>Row 2, Column 3</td>
</tr>
</table>

Divs

The DIV tag is used to group content together and apply styles to the group as a whole. Divs are commonly used to create sections of a web page.

Example:

<div class="section">
<h2>Section Heading</h2>
<p>This is the content of the section.</p>
</div>

Comments

Comments can be added to HTML code to provide notes or explanations. Comments are not displayed in the browser, and can only be seen by looking at the source code

Example:

<!-- This is a comment -->

Conclusion

HTML is a powerful tool for creating web pages, and the tags and attributes covered in this document are just the tip of the iceberg. By mastering HTML, you can create dynamic, engaging, and responsive web pages that look great on any device. Keep learning and experimenting, and you'll be on your way to becoming an HTML expert in no time!
